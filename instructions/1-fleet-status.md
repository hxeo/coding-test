# Part 1 — Fleet Status Counter

Build a status widget for a fleet management dashboard inside `src/1-fleet-status.component.ts`.

The widget tracks how many vehicles are in each of three states: **active**, **idle**, and **maintenance**. It should:

- Show the current count for each state and a running total
- Allow each count to be incremented individually
- Alert the user when the maintenance load is high
- Log a warning when that alert becomes active

If anything is unclear, ask.
